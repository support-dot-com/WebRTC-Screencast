WebRTC-Screencast
=================

WebRTC prototype for view only ScreenCast

Requirements
============

1. OpenFire
2. Jingle Nodes plugin
3. SIP (With an outbound proxy)
4. Asterisk
5. WebSockets Plugin for OpenFire


NOTE
====

OpenFire's Jingle Nodes require a Public IP, the same is advised for XMPP and SIP.

Please compile to coffeescript
